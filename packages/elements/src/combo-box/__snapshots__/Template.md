# `combo-box/Template`

## `Template Parts`

####   `Empty DOM has all required parts`

```html
<div part="input-wrapper">
  <input
    aria-expanded="false"
    autocomplete="off"
    part="input"
    role="combobox"
    type="text"
  >
  <div
    id="toggle-button"
    part="button button-toggle"
  >
    <ef-icon
      icon="down"
      part="icon icon-toggle"
    >
    </ef-icon>
  </div>
</div>

```
